# Hisabi - حسابي

A personal expense tracker built with Flutter, Clean Architecture, Riverpod, and Firebase Firestore.

## Tech Stack

- **Flutter 3.x** / Dart
- **Riverpod** — state management
- **GoRouter** — navigation
- **Firebase Firestore** — cloud database
- **fl_chart** — analytics charts
- **Clean Architecture** — domain / data / presentation layers
